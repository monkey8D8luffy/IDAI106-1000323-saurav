# FutureForward Wellness 🌿

A high-fidelity Streamlit prototype designed for the IB Career-related Programme (AI - Design Thinking).

## 🧠 Concept
FutureForward Wellness is a smart stress-relief ecosystem combining:
- AI-driven mood detection
- Smart relaxation pods
- Personalized wellness sessions

## 🎯 Features
- Liquid Glass UI (Glassmorphism)
- Simulated AI MoodSync Engine
- Voice + Manual Stress Check-In
- Immersive Breathing + Audio Experience
- Zen Garden Gamification
- Reflection & Stress Tracking Dashboard
